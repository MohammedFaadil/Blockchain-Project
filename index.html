Blockchain FAQ

What is Blockchain?

Blockchain is a decentralized digital ledger where data blocks are linked using cryptography and are immutable.

What is Proof of Work?

Proof of Work is a consensus mechanism requiring miners to solve a computational challenge to add a block securely.

What is a block's hash?

A hash is a unique cryptographic fingerprint for each block; it identifies data and links blocks.
